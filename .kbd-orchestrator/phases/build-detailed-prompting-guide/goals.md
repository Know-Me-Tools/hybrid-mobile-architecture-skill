# Goals

- Publish the comprehensive prompting guide as first-class source documentation and in the KnowMe documentation site, not as a summary or pointer.
- Provide ten complete scenario recipes with copyable staged prompts for discovery, Feynman learning, KBD assessment, research, planning, bounded implementation, verification, adversarial criticism, reflection, recovery, and stop conditions.
- Cover Codex, Claude Code, OpenCode, Kimi Code CLI, Google Antigravity, and Zed with actual harness setup, invocation, permissions, skills, MCP, autonomous-loop, evidence-capture, and handoff examples.
- Maintain a dated official-source model registry and generate producer, critic, research, mechanical-work, and long-context routing guidance without inventing unsupported models or capabilities.
- Teach Feynman, KBD, Karpathy, PMPO, Claude/headless, producer-critic, cross-model, skill-creation, and native-agent-creation loops with authority boundaries, budgets, retry limits, human checkpoints, and termination rules.
- Document the OpenAI Proxy native-agent case study and show when to generate a reusable skill versus a typed native agent.
- Verify every recipe has observable success criteria and that the complete guide is navigable, searchable, sanitized, and published through the branded documentation site.
